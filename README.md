# Employee Attrition Prediction & Retention Dashboard

## Project Overview
This project focuses on analyzing employee attrition using HR data and building an interactive retention dashboard using Excel and Power BI. The objective is to identify attrition patterns, calculate key HR KPIs, and highlight employees at high risk of leaving.

## Key Objectives
- Analyze employee attrition trends
- Identify high-risk employees using a rule-based risk scoring model
- Support data-driven HR retention strategies

## Dataset
The dataset includes:
- Employee demographics
- Department and job details
- Tenure, salary, training hours
- Performance ratings and overtime
- Attrition status

File: `data/employee_attrition_data.xlsx`

## Dashboard Features
- KPI Cards: Attrition Rate %, Average Tenure, Overtime %, High-Risk Employee Count
- Attrition by Department (Bar Chart)
- Attrition by Gender (Pie Chart)
- Training vs Performance vs Attrition (Scatter Plot)
- Top High-Risk Employees Table
- Interactive slicers (Department, Gender, Risk Level)

Dashboard Output:  
`dashboard/employee_attrition_dashboard.pdf`

## Tools Used
- Microsoft Excel (Data Cleaning, KPI Calculation)
- Power BI (Dashboard and Visual Analytics)
